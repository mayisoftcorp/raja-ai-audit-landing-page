# RAJA @ MAYISOFTCORP — AI Automation Audit Landing Page

A conversion-focused static landing page for the primary offer from the personal brand strategy:

**AI Automation Audit for SMBs**

## Strategy summary

- Target visitor: SMB founders, owners, and operators with repetitive operations, admin, sales-support, or customer-communication workflows.
- Core promise: identify practical Agentic AI automation opportunities that can save time, reduce friction, and improve business execution.
- Primary CTA: Message “AI Audit”.
- CTA destination: `mailto:gansivraj@gmail.com`.
- Design direction: sharp, modern, systems-oriented, founder-led, and practical.

## Files

- `index.html` — accessible static landing page
- `styles.css` — responsive visual design
- `assets/raja-mayisoft-brand.png` — brand visual from the personal brand folder
- `strategy.md` — extracted landing page strategy and QA notes
